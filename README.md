## ⚙️ วิธีการแพ็คไฟล์และติดตั้งม็อด (สำหรับ Source Code)
ในโฟลเดอร์นี้คือไฟล์ Source ของม็อดแปลภาษาไทย ที่คุณสามารถนำไปแพ็คเป็นไฟล์ `.pck` เพื่อนำไปใช้กับตัวเกมได้

**สิ่งที่ต้องเตรียม:**
1. โปรแกรม **Godot PCK Explorer** (ดาวน์โหลดและติดตั้งให้เรียบร้อย)
2. โฟลเดอร์ Source Code ของม็อดแปลภาษาไทยนี้

**ขั้นตอนการแพ็คไฟล์และติดตั้ง:**
1. เปิดโปรแกรม **Godot PCK Explorer** ขึ้นมา
2. เลือก **Create PCK** (หรือ Pack Folder)
3. เลือกโฟลเดอร์ม็อดนี้ (โฟลเดอร์ที่มีไฟล์ `Constants.gd`, `assets/`, `components/`, `data/` ฯลฯ) เพื่อทำการแพ็ค
4. ตั้งชื่อไฟล์ปลายทาง เช่น `UntilThen_Temp.pck` (หรือตั้งชื่อให้ตรงกับไฟล์ที่คุณต้องการนำไปวางทับ)
5. กำหนดเวอร์ชันให้ตรงกับของเกมเป็น **Godot 4.x**
6. กด **Pack** เพื่อสร้างไฟล์ `.pck`
7. เมื่อได้ไฟล์ `.pck` แล้ว ให้นำไฟล์นี้ไปวางทับในโฟลเดอร์ของเกม Until Then (ในโฟลเดอร์ที่มีไฟล์เกม `.exe`) 
*(⚠️ อย่าลืมสำรองไฟล์ `.pck` ต้นฉบับของเกมเก็บไว้ก่อน)*
8. เข้าเกมและสนุกกับเนื้อเรื่องภาษาไทยได้เลย!
