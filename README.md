## ⚙️ วิธีการติดตั้งม็อด
**สิ่งที่ต้องเตรียม:**
1. โปรแกรม **Godot PCK Explorer** (สำหรับแตกและแพ็คไฟล์เกม)
2. ไฟล์ม็อดภาษาไทยที่ดาวน์โหลดจากแท็บ [Releases] ของหน้านี้

**ขั้นตอนการติดตั้ง:**
1. เปิดโปรแกรม **Godot PCK Explorer** แล้วเลือกเปิดไฟล์หลักของตัวเกม (`.pck`) จากนั้นทำการ Extract ดาต้าทั้งหมดออกมาไว้ในโฟลเดอร์บนคอมพิวเตอร์ของคุณ
2. แตกไฟล์ `.zip` ของม็อดภาษาไทยที่ดาวน์โหลดไป นำไฟล์ภาษาที่ได้ไปวางทับในโฟลเดอร์ `extracted/assets/story/locales`
3. ใช้โปรแกรม **Godot PCK Explorer** แพ็คโฟลเดอร์ทั้งหมดกลับไปเป็นไฟล์ `.pck` อีกครั้ง
4. นำไฟล์ `.pck` ใหม่ที่ได้ ไปวางทับไฟล์เดิมในโฟลเดอร์ติดตั้งเกม *(⚠️ อย่าลืมสำรองไฟล์ต้นฉบับเก็บไว้ก่อน)*
5. เข้าเกมและสนุกกับเนื้อเรื่องภาษาไทยได้เลย!
