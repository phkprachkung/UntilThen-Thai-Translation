## ⚙️ วิธีการติดตั้งม็อด
**สิ่งที่ต้องเตรียม:**
1. โปรแกรม **Godot PCK Explorer** (สำหรับแตกและแพ็คไฟล์เกม)
2. ไฟล์ม็อดภาษาไทยที่ดาวน์โหลดจากแท็บ [Releases]

**ขั้นตอนการติดตั้ง:**
1. ใช้ **Godot PCK Explorer** แตกไฟล์หลักของตัวเกม (`.pck`) ออกมาไว้ในโฟลเดอร์
2. แตกไฟล์ `.zip` ของม็อดภาษาไทย นำไฟล์ที่ได้ไปวางทับใน `extracted/assets/story/locales`
3. ใช้ **Godot PCK Explorer** แพ็คโฟลเดอร์ทั้งหมดกลับไปเป็นไฟล์ `.pck`
4. นำไฟล์ `.pck` ไปวางทับไฟล์เดิมในโฟลเดอร์ติดตั้งเกม (อย่าลืมสำรองไฟล์ต้นฉบับไว้ก่อน)
5. เข้าเกมและเปลี่ยนภาษาในเมนูตั้งค่า
